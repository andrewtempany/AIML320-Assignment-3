# AIML320 Assignment 3 — Part 2

Two notebooks: [`perceptron.ipynb`](perceptron.ipynb) (Tasks a & b) and
[`mlp.ipynb`](mlp.ipynb) (Task c).

## Requirements

- Python 3.11+
- Jupyter Notebook (or an equivalent `.ipynb` runtime — JupyterLab,
  VS Code with the Python extension, PyCharm, etc.)

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Run

```bash
jupyter notebook
```

Open either notebook and **Run All Cells**. CSVs are read from `data/`, so
launch from the project root.
